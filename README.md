# tiktokyoutube_analysis
An end-to-end data pipeline and analysis dashboard for TikTok and YouTube short video trends.
## About The Project
This is an end-to-end data pipeline and analysis dashboard designed to track and analyze short video trends across TikTok and YouTube. It processes raw data and visualizes key metrics to provide insightful analytics.

## Built With
* **Backend:** Python, Flask
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite
* **Deployment:** Render (Cloud Hosting)
